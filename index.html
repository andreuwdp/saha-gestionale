Come ottenere le credenziali
1. Vai su Admin Shopify → Impostazioni → App e canali di vendita
2. Clicca "Sviluppa app" → "Crea un'app"
3. Vai su "Configurazione API" → "Admin API"
4. Abilita i permessi: read_products, write_products, read_inventory, write_inventory, read_orders
5. Clicca "Installa app" → copia il token
Il token viene salvato solo nel tuo browser (localStorage). Non viene mai inviato a server esterni.
